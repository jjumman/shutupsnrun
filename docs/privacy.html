개인정보 보호정책

최종 업데이트: 2024년 11월

지원 개인정보 보호정책

닥치고뛰어("앱")는 사용자의 개인정보를 중요하게 생각하며, 최소한의 데이터만 수집하고 처리합니다. 본 개인정보 보호정책은 앱이 어떤 정보를 수집하고, 어떻게 사용하며, 어떻게 보호하는지 설명합니다.

1. 개요

닥치고뛰어 앱(이하 "앱")은 ShutUpAndRun Team(이하 "개발자")이 제공하는 러닝 트래킹 애플리케이션입니다.

개발자는 사용자의 프라이버시를 존중하며, 다음 원칙을 따릅니다:

  • 최소한의 정보만 수집합니다
  • 수집한 정보는 기기 내에만 저장합니다
  • 사용자 정보를 제3자와 공유하지 않습니다
  • 로그인이나 계정 생성이 필요하지 않습니다

2. 수집하는 정보

2.1 위치 정보

수집 목적: 러닝 활동 추적 및 기록

수집 방법: GPS를 통한 실시간 위치 추적

수집 시점: 사용자가 "러닝 시작" 버튼을 눌렀을 때부터 "완료" 버튼을 누를 때까지

저장 위치: 사용자 기기의 로컬 데이터베이스에만 저장됩니다. 서버로 전송되지 않습니다.

사용 범위:

  • 러닝 경로 표시
  • 이동 거리 계산
  • 현재 위치 지도 표시

2.2 사진 (선택 사항)

수집 목적: 프로필 사진 설정

수집 방법: 사용자가 프로필 사진을 직접 선택할 때

저장 위치: 사용자 기기의 앱 내부 저장소에만 저장됩니다.

참고: 사진 선택은 완전히 선택 사항이며, 앱의 핵심 기능 사용에 필수가 아닙니다.

2.3 앱 사용 데이터

앱은 다음 데이터를 기기 내에 저장합니다:

  • 러닝 기록 (날짜, 시간, 거리, 페이스, 경로)
  • 사용자 이름 (사용자가 직접 입력한 경우)
  • 프로필 사진 (사용자가 직접 선택한 경우)

중요: 이 모든 데이터는 사용자의 기기에만 저장되며, 외부 서버로 전송되거나 백업되지 않습니다.

3. 정보의 사용

수집된 정보는 다음 목적으로만 사용됩니다:

  • 위치 정보: 러닝 경로 추적, 거리 계산, 지도 표시
  • 사진: 프로필 화면에 표시
  • 러닝 기록: 사용자의 러닝 히스토리 표시 및 통계

개발자는 사용자의 개인 정보에 접근할 수 없으며, 모든 데이터는 사용자의 기기에만 저장됩니다.

4. 정보의 공유 및 제3자 제공

앱은 사용자의 개인 정보를 제3자와 공유하지 않습니다.

4.1 제3자 서비스

앱은 다음 제3자 서비스를 사용합니다:

Google AdMob (광고 서비스)

  • 목적: 앱 내 광고 표시
  • 수집 정보: 광고 ID, 기기 정보, 광고 상호작용 데이터
  • 개인정보 보호정책: Google 개인정보 보호정책
  • 광고 맞춤설정 해제: iOS 설정 > 개인정보 보호 > 추적 > 앱이 추적을 요청하도록 허용 끄기

참고: AdMob은 광고 표시를 위해 기기 정보를 수집할 수 있지만, 개발자는 이 정보에 접근할 수 없습니다.

5. 정보의 보관 및 삭제

5.1 보관 기간

모든 데이터는 사용자가 앱을 삭제할 때까지 기기에 보관됩니다.

5.2 데이터 삭제

사용자는 언제든지 다음 방법으로 데이터를 삭제할 수 있습니다:

  • 개별 러닝 기록 삭제: 전체 러닝 기록 화면에서 기록을 왼쪽으로 스와이프
  • 프로필 사진 삭제: 프로필 화면에서 사진을 다시 선택하거나 앱 재설치
  • 모든 데이터 삭제: 기기에서 앱을 삭제하면 모든 데이터가 완전히 삭제됩니다

6. 데이터 보안

개발자는 사용자 데이터 보호를 위해 다음 조치를 취합니다:

  • 모든 데이터는 기기 내부에만 저장됩니다
  • 외부 서버로 데이터를 전송하지 않습니다
  • iOS의 보안 샌드박스 내에서 데이터를 보호합니다
  • 다른 앱이나 사용자는 앱의 데이터에 접근할 수 없습니다

7. 아동의 개인정보 보호

본 앱은 만 4세 이상의 모든 연령층이 사용할 수 있습니다. 앱은 의도적으로 만 14세 미만 아동의 개인 식별 정보를 수집하지 않습니다.

만약 부모님이나 보호자가 자녀가 개인 정보를 제공했다는 것을 알게 되면, jjumman@gmail.com으로 연락주시기 바랍니다.

8. 권한 안내

앱이 요청하는 권한과 그 사용 목적은 다음과 같습니다:

8.1 필수 권한

  • 위치 (Location) - "앱 사용 중"

    러닝 경로 추적 및 거리 측정을 위해 필요합니다.

  • 위치 (Location) - "항상"

    백그라운드에서도 러닝을 지속적으로 추적하기 위해 필요합니다. 러닝 중에만 활성화됩니다.

8.2 선택 권한

  • 사진 (Photos)

    프로필 사진 설정을 위해 사용됩니다. 거부하셔도 앱의 핵심 기능을 사용하는 데 문제가 없습니다.

9. 사용자의 권리

사용자는 다음과 같은 권리를 가집니다:

  • 접근권: 언제든지 앱 내에서 자신의 데이터를 확인할 수 있습니다
  • 수정권: 프로필 정보를 언제든지 수정할 수 있습니다
  • 삭제권: 개별 러닝 기록이나 앱 전체를 삭제할 수 있습니다
  • 거부권: 선택적 권한(사진)을 거부할 수 있습니다

10. 개인정보 보호정책의 변경

본 개인정보 보호정책은 법률 변경, 서비스 업데이트 등에 따라 수정될 수 있습니다.

정책이 변경되는 경우:

  • 본 페이지에 업데이트된 정책을 게시합니다
  • "최종 업데이트" 날짜를 변경합니다
  • 중요한 변경사항이 있는 경우 앱 내 공지를 통해 알립니다

정책 변경 후에도 앱을 계속 사용하시는 경우, 변경된 정책에 동의하는 것으로 간주됩니다.
